# 💼 Human Resources (HR) Management System

تطبيق متكامل لإدارة الموارد البشرية والموظفين، تم بناؤه باستخدام أحدث تقنيات الويب لتقديم تجربة مستخدم سلسة وسريعة ومخصصة بالكامل.

## 🚀 التقنيات المستخدمة (Tech Stack)
* **React.js** (v18)
* **TypeScript** (لضمان كود قوي، منظم وخالي من الأخطاء)
* **Tailwind CSS** (للتصميم الأنيق، العصري والمتجاوب بالكامل)
* **Vite** (كأداة بناء وتطوير فائقة السرعة)
* **Lucide React & Framer Motion** (للأيقونات والحركات الإنسيابية داخل النظام)

## 🌟 المميزات الرئيسية
* **لوحة تحكم خاصة بالمدير (Manager Dashboard):** لمتابعة طلبات الموظفين، الإجازات، وعرض الإحصائيات العامة للشركة.
* **لوحة تحكم خاصة بالموظف (Employee Dashboard):** لتقديم طلبات الإجازة، متابعة المهام اليومية، وتعديل البيانات الشخصية.
* **نظام صلاحيات مخصص:** يوجه كل مستخدم للوحة التحكم الخاصة به فور تسجيل الدخول بنجاح.
* **دعم كامل للـ RTL:** واجهة مستخدم متوافقة تماماً مع اللغة العربية (من اليمين إلى اليسار).
